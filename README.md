# South Africa Crime Analysis

> ⚠️ **Work in Progress** — this project is actively being developed.

## Overview
A data science project examining over a decade of crime in South Africa 
(2011–2023). Analyses patterns across crime categories and regions using R.

## Methods
- Exploratory Data Analysis (EDA)
- Time Series Analysis
- Statistical Inference
- Data Visualisation

## Repository Structure
- `data/` — raw and processed datasets
- `scripts/` — R scripts for analysis
- `outputs/` — plots and results

## Notes
Part of an ongoing data analytics portfolio.
